HeuristiCheck is a professional Chrome Extension designed for UX Designers, Developers, and QA Testers. It performs an automated audit of any webpage, identifying usability and accessibility violations based on Jakob Nielsen's 10 Usability Heuristics. It helps you identify, navigate, and fix UX issues with AI-powered suggestions and agency-quality reporting.

## ðŸš€ Installation Guide

1.  **Download** this project folder to your computer.
2.  Open Google Chrome and navigate to `chrome://extensions/`.
3.  Enable **Developer mode** using the toggle switch in the top-right corner.
4.  Click the **Load unpacked** button in the top-left corner.
5.  Select the root folder of **HeuristiCheck**.
6.  The extension is now installed! Pin it to your toolbar for easy access.

## ðŸ“– How to Use

1.  **Navigate** to any website you want to audit.
2.  Click the **HeuristiCheck icon** in the browser toolbar.
3.  Click **Start Evaluation**.
4.  **View Results:**
    *   **Popup:** See a high-level overview of passed/failed heuristics.
    *   **On-Page:** Use the floating widget to click through specific errors on the page.
5.  **Use AI Features:**
    *   Click the **Settings (Gear)** icon in the popup.
    *   Enter your **OpenAI API Key**.
    *   In the on-page widget, click the **Magic Wand (âœ¨)** icon to get AI suggestions.
6.  **Export Report:** Click **Download PDF** to generate a professional audit report.
